Curriculum

Five Core Modules

Each module is a self-contained, hands-on unit with labs on real AWS infrastructure — not simulated environments.

01
Module 01 · Foundation
☁️
Phase 1
Cloud Computing Fundamentals

Build the mental model every cloud practitioner needs. Understand how cloud providers work, what you pay for, and why cloud has replaced on-premise infrastructure in modern engineering.

  • IaaS, PaaS & SaaS service models explained
  • Cloud economics: CapEx vs OpEx, pay-as-you-go
  • AWS global infrastructure: Regions & Availability Zones
  • Shared Responsibility Model & compliance basics
  • AWS Management Console, CLI & SDK overview
⏱ Weeks 1–2 · ~8 hrs Beginner
02
Module 02 · AWS Core
⚙️
Phase 2
AWS Compute, Storage & Networking

Get hands on with the services that power 70% of cloud workloads. Launch EC2 instances, configure secure networks, store objects in S3, and lock everything down with IAM.

  • EC2: instance types, AMIs, user-data, key pairs
  • S3: buckets, policies, versioning & static hosting
  • VPC: subnets, route tables, internet & NAT gateways
  • Security Groups & NACLs for layered defence
  • IAM: users, roles, policies & least-privilege design
⏱ Weeks 3–4 · ~10 hrs Intermediate
03
Module 03 · Containers
🐳
Phase 3
Docker & Containerisation on AWS

Package your applications once, run them anywhere. Master Docker from first principles and integrate containers natively into the AWS ecosystem using ECR and ECS.

  • Docker architecture: images, containers & volumes
  • Writing production-grade Dockerfiles & .dockerignore
  • Docker Compose for multi-container local dev
  • Amazon ECR: building, tagging & pushing images
  • Introduction to ECS & Fargate for managed containers
⏱ Weeks 5–6 · ~10 hrs Intermediate
04
Module 04 · Deployment
🌐
Phase 4
Web Deployment, SSL & Serverless

Take real applications live. Configure Nginx as a reverse proxy on EC2, automate SSL with Let's Encrypt, build serverless APIs with Lambda, and connect services with event-driven patterns.

  • Nginx reverse proxy & virtual host configuration
  • SSL/TLS with Certbot + Let's Encrypt on EC2
  • Route 53: DNS, A/CNAME records & health checks
  • AWS Lambda: writing, deploying & testing functions
  • API Gateway: REST APIs, triggers & CORS setup
⏱ Weeks 7–8 · ~10 hrs Intermediate
05
Module 05 · Architecture
🏗️
Phase 5 · Capstone
Cloud Architecture, Databases & Certification Prep

Tie everything together. Design scalable, fault-tolerant architectures using AWS Well-Architected Framework pillars. Add managed databases, caching, and monitoring — then build a complete capstone project from scratch.

  • RDS (PostgreSQL / MySQL) setup, backups & replicas
  • ElastiCache & DynamoDB for high-performance data
  • Load Balancers & Auto Scaling Groups
  • CloudWatch metrics, alarms & logging
  • Well-Architected Framework: 6 pillars review
  • SAA-C03 exam strategy & mock assessment
⏱ Weeks 9–10 · ~12 hrs Advanced
