Why This First

Version control is the
first professional skill.

Every other course involves code. That code needs to be tracked, backed up, collaborated on, and shipped. Git is how that happens. Employers expect it from day one.

01
Never Lose Work Again
Git creates a complete history of every change. Roll back mistakes, compare versions, and work fearlessly knowing you can always recover.
02
Collaborate Like a Pro
Branching and merging let multiple people work on the same project simultaneously — without overwriting each other. This is how every real team works.
03
GitHub is Your Portfolio
Every project you build in every other course goes on GitHub. Recruiters look at your commit history. A green contribution graph speaks louder than a resume.
04
CI/CD and Automation
GitHub Actions lets your code test, build, and deploy itself. You'll use it in the AWS and Web Development courses — but it starts here.
Core Commands

What You'll Master

You won't just memorise commands — you'll understand why each one exists and build muscle memory through real projects.

Setup & Init
Repository Basics
Create repos, configure identity, understand the working tree, staging area, and commit history.
git init · git clone · git config
Daily Workflow
Add, Commit, Push
Stage changes selectively, write meaningful commit messages, and push to remote repositories.
git add · git commit · git push
Branching
Branches & Merging
Feature branches, merge strategies (fast-forward vs three-way), resolving conflicts, and rebase basics.
git branch · git merge · git rebase
Collaboration
Pull Requests & Reviews
Fork workflows, opening PRs, reviewing code, requesting changes, and squash merges.
gh pr create · git fetch · git pull
History & Debugging
Log, Diff & Blame
Navigate commit history, compare file versions, find who changed what and when using git blame and bisect.
git log · git diff · git blame
Advanced
Stash, Tags & GitHub Actions
Temporarily stash work, tag releases, and write a basic CI pipeline that runs tests on every push.
git stash · git tag · gh workflow
