Memory Model

You'll actually understand this.

Most courses skip over memory. This one makes it the centrepiece — because everything else flows from it.

Stack — grows downward
0x7fff...f0
main()
← current frame
0x7fff...d0
int x = 42

0x7fff...c8
int *p
← pointer on stack
Heap — grows upward
0x55...1000
malloc(100)
← *p points here
