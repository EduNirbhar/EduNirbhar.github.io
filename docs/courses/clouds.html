Learning Path

Your Road to AWS Mastery

Five progressive phases — engineered so each week builds directly on the last, from cloud theory to certified practitioner.

AWS Core: Compute & Networking

Curriculum

Five Core Modules

Each module is a self-contained, hands-on unit with labs on real AWS infrastructure — not simulated environments.

☁️

Cloud Fundamentals

Build the mental model every cloud practitioner needs. Understand how cloud providers work, what you pay for, and why cloud has replaced on-premise infrastructure.

IaaS, PaaS & SaaS Cloud economics AWS global infrastructure Shared Responsibility
⚙️

AWS Core Services

Get hands on with the services that power 70% of cloud workloads. Launch EC2 instances, configure secure networks, store objects in S3, and lock down IAM.

EC2 & AMIs S3 policies VPC & subnets IAM roles
🐳

Docker & Containers

Package your applications once, run them anywhere. Master Docker from first principles and integrate containers natively into AWS using ECR and ECS.

Docker architecture Dockerfiles Amazon ECR ECS Fargate

Web & Serverless

Take real applications live. Configure Nginx on EC2, automate SSL with Let's Encrypt, build APIs with Lambda, and connect services with event patterns.

Architecture & Cert Prep

Tie everything together. Design scalable architectures using AWS Well-Architected Framework pillars. Add managed databases, caching, and monitoring — then build a complete capstone project.

RDS backups & replicas Load Balancers Auto Scaling CloudWatch SAA-C03 mock tests

Hands-On Projects

Build Real Things

Every module culminates in a project deployed to live AWS infrastructure — not sandboxed demos.

Project 01

Secure Multi-Tier VPC

Design and deploy a VPC with public/private subnets, NAT gateways, bastion host, and security group rules.

Project 02

App on ECS Fargate

Dockerise a web application, push to ECR, and deploy as a managed ECS Fargate service behind an ALB.

Project 03

Serverless REST API

Build an API using Lambda + API Gateway backed by DynamoDB, with IAM-secured endpoints.

Project 04

Global Static Website

Host a site on S3, front it with CloudFront CDN, attach an ACM SSL certificate, and configure Route 53.

Project 05

Auto-Scaling Web App

Deploy an app across multiple AZs with an ALB, Auto Scaling Group, RDS Multi-AZ, and CloudWatch alarms.

Capstone

Production Platform

Synthesise all modules into a single full-stack platform: containers, databases, monitoring, and CI/CD.
